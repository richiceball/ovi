SISTEMA CONTROL DE VERSIONES (VCS)

VCS, por sus siglas en inglés, Version Control System, es un sistema que permite registrar todos los cambios que se han realizado sobre un conjunto de archivos durante el ciclo de vida de desarrollo de un aplicativo informático. Aunque puede ser usado con otros fines, pero en este artículo se destacará su importancia en función al desarrollo de software.

Este sistema permite navegar durante los diferentes cambios realizados, lo cual es una herramienta muy útil para el seguimiento y corrección de errores, y absolutamente necesario para el trabajo en equipo.

Tipos de sistemas control de versiones

  1. Sistema local
  2. Sistema centralizado
  3. Sistema distribuido
